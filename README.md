# Remote-Work-Productivity-Predictor

This app is based on a Linear Regression model, which takes the following parameters as input from the users:
1. Age
2.  Years of Experience,
3.  Number of Daily Meetings,
4.  Number of Hours Worked per Day,
5.  Internet Speed (Mbps),
6.  Home Office Score (Work Environment out of 10)
7.  Number of Distractions per Day

## Productivity Score Interpretation:

Below 20: Low Productivity Score
20-25: Average Productivity Score
25-30: Good Productivity Score
Above 30: Excellent Productivity Score
